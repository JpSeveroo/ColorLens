Sobre / Ajuda

Entenda como o ColorLens funciona e como tirar o máximo proveito.

ColorLens é uma ferramenta projetada para ajudar pessoas com daltonismo e outras dificuldades de visão a navegar na web com mais conforto e clareza.

Use os filtros para ajustar as cores das páginas, crie perfis personalizados para mapear cores específicas e utilize as ferramentas de acessibilidade para melhorar sua experiência de leitura.


Como Usar o ColorLens


Filtros de Daltonismo: Escolha um tipo de daltonismo para que a extensão aplique uma correção de cor em tempo real a todas as páginas.

Personalização de Cores:Ideal para casos específicos onde você precisa diferenciar cores que parecem iguais. Mapeie uma cor de origem (ex: vermelho) para uma cor de destino (ex: azul) que você consiga ver bem.

Acessibilidade Visual: Ajuste o contraste e a saturação de forma global ou ative modos especiais como o de leitura ou noturno para reduzir o cansaço visual.
